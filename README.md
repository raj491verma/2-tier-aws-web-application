# 2-tier-aws-web-application
# Architecture
This project demonstrates a simple 2-tier web application using AWS sevices like EC2 RDS.

## Tier 1: Web Server
-EC2(Amazon Linux)
-Apache/httpd
-Python

## Tier 2: Database
-Amazon RDS (Mysql)

## Flow
User->EC2 Web Server->RDS Database

## AWS Services Used
-EC2
-RDS
-VPC
-Security Groups

## How to Deploy
1.launch EC2
2.Install web server
3.Create RDS database
4.Connect EC2 to RDS
5.Deploy application Code
